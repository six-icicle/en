How to read these. Each card simulates the actual app shell — navbar with hinomaru brand mark, an active tile with a faux claude welcome, an empty stub, and a foot bar — using only that variant's palette tokens. The token strip below maps to en's existing --bg / --tile / --tile-2 / --ink / --ink-2 / --accent / --ok / --warn contract, so picking a winner is one find/replace away from a working theme.

Two axes to consider. Temperature (warm → cool: akari → sumi-yoru → yamizumi → beni-yoru) and accent restraint (most → least muted: yamizumi → akari → sumi-yoru → beni-yoru). Sumi Yoru is the safest default; Yamizumi is the most "tea-ceremony"; Akari leans warm/cozy; Beni Yoru is closest to the current rose-pine-soft default in feel.
