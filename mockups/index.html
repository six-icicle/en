Hub for many concurrent Claude CLI sessions. Layout direction landed on the live grid (25). Below: palette variations of that layout, plus earlier metaphor exploration rounds for reference.

Live grid — every session visible, ⌘+arrow to focus
User pivoted away from the dock+main-pane model. All sessions tile in a 2×2 grid; one is focused, the others are dimmed but readable. Hyper-modern, minimal chrome.
